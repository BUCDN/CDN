Github 开启 jsDelivr 加速的方法

1、源码上传 GitHub 仓库

2、链接按如下格式修改

https://cdn.jsdelivr.net/gh/用户名/仓库名/仓库路径/文件名

https://cdn.jsdelivr.net/gh/用户名/仓库名@版本号/仓库路径/文件名

https://cdn.jsdelivr.net/gh/用户名/仓库名@latest/仓库路径/文件名

https://cdn.jsdelivr.net/gh/bucdn/cdn@latest/css/normalize/801/normalize.css

3、获取目录列表

https://cdn.jsdelivr.net/gh/用户名/仓库名@版本号/

https://cdn.jsdelivr.net/gh/用户名/仓库名@版本号/dist/

https://cdn.jsdelivr.net/gh/bucdn/cdn@latest/

4、清理缓存

将引用链接里的 "cdn" 改成 "purge"

https://purge.jsdelivr.net/gh/用户名/仓库名@latest/仓库路径/文件名

https://purge.jsdelivr.net/gh/bucdn/cdn@latest/css/normalize/801/normalize.css

5、查看 CDN 上的 tags 和 versions

将引用链接 "cdn.jsdelivr.net/gh" 改成 "data.jsdelivr.com/v1/package/gh"

https://data.jsdelivr.com/v1/package/gh/bucdn/cdn@latest/
